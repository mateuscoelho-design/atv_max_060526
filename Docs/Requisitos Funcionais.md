**RF01 — Cadastro de Cinemas**

O sistema deve permitir o cadastro de cinemas contendo nome, endereço, cidade e capacidade de público. 

---

**RF02 — Cadastro de Filmes**

O sistema deve permitir o cadastro de filmes contendo título, gênero, duração, diretor e elenco. 

---

**RF03 — Gerenciamento de Sessões**

O sistema deve permitir cadastrar sessões vinculando um filme a um cinema, informando data e horário. 

---

**RF04 — Consulta de Filmes em Cartaz**

O sistema deve permitir consultar os filmes disponíveis em cada cinema. 

---

**RF05 — Registro de Público**

O sistema deve permitir registrar a quantidade de espectadores presentes em cada sessão.

---

**RF06 — Consulta de Informações dos Filmes**

O sistema deve permitir visualizar informações detalhadas dos filmes, incluindo gênero, diretor e elenco.
